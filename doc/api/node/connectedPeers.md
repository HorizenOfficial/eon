### node/connectedPeers

Returns the list of node peers currently connected to this node.

**Parameters**

No parameters

**Example request**

     curl -sX POST 'http://127.0.0.1:9085/node/connectedPeers' -H 'Content-Type: application/json' -H 'accept: application/json'
