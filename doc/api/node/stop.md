### node/stop

Initiates a graceful stop procedure for the sidechain node. Returns an empty object.

**Parameters**

No parameter

**Example request**

     curl -sX POST 'http://127.0.0.1:9085/node/stop' -H 'Content-Type: application/json' -H 'accept: application/json' 

**Example response**

    {
        "result" : {}
     }
