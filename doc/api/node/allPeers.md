### node/allPeers

Returns the list of all sidechain nodes known by this node.

**Parameters**

No parameters

**Example request**

     curl -sX POST 'http://127.0.0.1:9085/node/allPeers' -H 'Content-Type: application/json' -H 'accept: application/json'
